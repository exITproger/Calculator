# Calculator
Консольное приложение «Калькулятор» для выполнения арифметических операций над числами с выводом результата во всех основных системах счисления.
## Основные возможности
### 1. Выбор системы счисления
При запуске программы пользователь выбирает одну из доступных систем счисления:

  - HEX — шестнадцатеричная

  - DEC — десятичная

  - OCT — восьмеричная

  - BIN — двоичная

### 2. Ввод выражения
После выбора системы пользователь вводит математическое выражение, используя числа в выбранной системе.

### 3. Поддерживаемые операции
Калькулятор обрабатывает следующие операции:

  - Сложение	+
  - Вычитание	-
  - Умножение	*
  - Деление	/
  - Остаток от деления	%

Программа корректно работает с положительными и отрицательными числами.

### 4. Обработка выражения
Приложение выполняет следующие шаги:

Проверка корректности введённого выражения

Преобразование чисел в десятичную систему (при необходимости)

Вычисление результата

### 5. Вывод результата
Результат вычисления отображается одновременно во всех системах счисления:

  - HEX (шестнадцатеричная)

  - DEC (десятичная)

  - OCT (восьмеричная)

  - BIN (двоичная)
